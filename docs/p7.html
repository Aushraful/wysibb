WysiBB language

To change language, you must add language file to the page.

<script src="/js/jquery.wysibb.min.js"></script>
<link rel="stylesheet" href="/css/default/wbbtheme.css" type="text/css" />

<script src="/js/lang/fr.js"></script>

And set lang in config

<script>
$(document).ready(function() {
 var wbbOpt = {
  lang: "fr"
 }
 $("#editor").wysibb(wbbOpt);
});
</script>
.....
<textarea id="editor"></textaera>
